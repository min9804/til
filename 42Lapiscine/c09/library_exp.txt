gcc를 통해 오브젝트 파일을 만든다.

*정적 라이브러리 만들기
-ar 이라는 명령어를 이용해서 라이브러리 아카이브 파일을 만듬

*ar옵션
c : 아카이브가 존재하지 않은 경우 생성 하라는 옵션
r : 아카이브에 새로운 오브젝트를 추가한다는 옵션

ex)
gcc -c -Wall -Wextra -werror *.c
ar -cr libft.a *.o

라이브러리를 사용하여 컴파일 예시)
gcc -o print_sum print_sum.c -L./-ft

-L 옵션은 라이브러리 위치를 알려준다.
표준 라이브러리의 경우 -L을 사용하지 않아도 된다.
